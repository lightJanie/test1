#### src文件夹 -代码文件夹

```
----assets文件夹：存储项目中自己的一些静态文件。
----components文件夹：存储项目中的自定义组件（小组件，公共组件）
----views 文件夹：村春项目中的自定义组件（大组件，页面级组件，路由级别组件）
----router 文件夹：存储VueRouter相关组件
----store文件夹：存储Vuex相关组件
----App.vue:跟组件
----main.js:入口文件
```
